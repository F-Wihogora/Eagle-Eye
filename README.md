Eagle Eye — Smart Public Security Command System
🛡️ Overview

Eagle Eye is a futuristic smart public security and emergency response command system designed to help law enforcement and public safety agencies monitor, predict, and respond to incidents in real time.

The platform combines:

📡 AI-powered surveillance
🗺️ Live incident mapping
📊 Crime prediction analytics
🚔 Emergency dispatch management
🧾 Citizen incident reporting

Built with pure HTML, CSS, and JavaScript, the system delivers a modern cyber-security styled dashboard focused on situational awareness and operational control.
